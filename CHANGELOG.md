## [0.0.14] - 2025-11-07

### Added
- Mapping file support for image de-identification
- Checkbox to toggle applying of default filter script
- Input labels for better form accessibility

### Changed
- Update CLI and UI for mapping file changes
- Update de-identification profile documentation
- Output directory structure improvements
- Save everything as XLSX format
- Improved project delete functionality

### Fixed
- Disable buttons when form is invalid
- Fixed failing tests
- Date window configuration properly passed to config

### Security
- Bump urllib3 from 2.4.0 to 2.5.0
- Bump npm dependencies across directories
